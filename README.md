# BSC
Front End developer v BSC, verze pro rok 2015

# Instalace

1. `git clone https://github.com/evaluecz/bsc.git`<br/>
2. `cd bsc`<br/>
3. `npm install`

# Spuštění aplikace
`npm start app.js`<br/>
Aplikace pak běží na http://localhost:900

# Protractor test
V adresáři `test`<br/>
`protractor test/protractor-cfg.js`
